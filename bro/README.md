# 🏰 Presentación: Transformación Digital de Disney

Este directorio contiene los archivos para la presentación sobre la **Transformación Digital de The Walt Disney Company** tras la adquisición de 21st Century Fox.

## 📁 Contenido del Directorio

### 📄 Archivos de Presentación

1. **`presentacion-disney.html`** - Presentación HTML interactiva
   - Utiliza Reveal.js para navegación fluida
   - Diseño responsivo y profesional
   - Colores corporativos de Disney
   - 5 diapositivas enfocadas en elementos visuales

2. **`slides.md`** - Presentación en formato Markdown
   - Compatible con GitHub Pages
   - Formato estándar para reveal-md
   - Fácil de editar y mantener
   - Exportable a múltiples formatos

3. **`generate_presentation.py`** - Script Python para generar PowerPoint
   - Genera archivo `.pptx` profesional
   - Utiliza biblioteca `python-pptx`
   - Colores y diseño corporativo Disney
   - Automatización completa del proceso

4. **`Disney_Transformacion_Digital_Presentacion.pptx`** - Archivo PowerPoint generado
   - Listo para usar en presentaciones
   - 5 diapositivas optimizadas
   - Diseño profesional y visual

## 🚀 Cómo Usar los Archivos

### 📱 Presentación HTML (presentacion-disney.html)
```bash
# Abrir en navegador web
open presentacion-disney.html
# o servir localmente
python3 -m http.server 8000
# Luego navegar a http://localhost:8000/presentacion-disney.html
```

**Controles de navegación:**
- ➡️ Flecha derecha: Siguiente diapositiva
- ⬅️ Flecha izquierda: Diapositiva anterior
- ESC: Vista general de todas las diapositivas

### 📝 Presentación Markdown (slides.md)
```bash
# Con reveal-md (requiere instalación)
npm install -g reveal-md
reveal-md slides.md

# Con pandoc (para convertir a otros formatos)
pandoc slides.md -o presentation.pdf
```

### 🐍 Script Python (generate_presentation.py)
```bash
# Instalar dependencias
pip install python-pptx

# Ejecutar script
python3 generate_presentation.py

# Resultado: Disney_Transformacion_Digital_Presentacion.pptx
```
